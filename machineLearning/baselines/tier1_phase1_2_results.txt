Tier 1 — Phase 1 (Bootstrap 95% CIs) + Phase 2 (5-fold Stratified CV)
=====================================================================
Run date: 2026-05-06
Pipeline state: post-2026-05-05 retrain (commit 9bd0b6d, AGE>=18 cohort,
                Index_LOS, isotonic-calibrated XGB champions)

Method:
- Bootstrap: 1,000 resamples-with-replacement of the held-out test set's
  predicted probabilities (seed=42). Reports point estimate + 2.5/97.5
  percentile band.
- 5-fold CV: StratifiedKFold(n_splits=5, shuffle=True, random_state=42)
  over the FULL cohort. Each fold re-fits the ColumnTransformer + each
  model with the production hyperparameters; XGB is re-calibrated inside
  each fold via CalibratedClassifierCV(method='isotonic', cv=5).
  Threshold = F1-max learned on TRAIN proba per fold.
